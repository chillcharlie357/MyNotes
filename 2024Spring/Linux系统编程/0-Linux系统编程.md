## 硬盘组织方式

- 硬盘组织方式：决定分区在磁盘上怎么组织
	1. MBR: Master Boot Record
		1. 只能有4个主分区，1个主分区可以变成4个逻辑分区$
		2. 最大支持4T
	2. GPT: GUID Partition Table Scheme
		1. 会浪费空间
		2. 支持4T以上


## 文件系统

- 文件系统
	- OS中负责存取和管理文件的部分，决定数据在分区里怎么放
	- Linux中常见：EXT4
	- Windows：NTFS

每个系统对自己的文件系统支持最好
