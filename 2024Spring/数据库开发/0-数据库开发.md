# 开发成功数据库的要点

1. 理解数据库体系结构
2. 理解锁和并发控制特性
3. 不要把数据库当黑盒
4. 性能、安全性都是适当的被设计出来的
5. 用尽可能简单的方法解决问题
6. DBA和RD之间的关系
