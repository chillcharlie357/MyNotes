# 调式
- 调式：以一个音为中心其他按照一定规律安排起来，形成体系
- 结尾在哪个音上，调式就是这个音

## 大调式
- do 明亮 灿烂 **大调式**
	- 南大校歌
	- **国歌**   
		- 词：田汉 曲：聂耳
		- 标准版47秒。不少于45秒，不大于50秒/
		- 1982年宪法
		- 鲜红的太阳永不落->隔壁太阳旗，不合适
- 自然大调：没有变调
- 和声大调：降la 
	- 《我的太阳》 
	- 最明亮辉煌
- 旋律大调：降la si 
	- 相对柔和

# 巴洛克时期:17-18世纪

## 特征

- 巴洛克:
	- 来自建筑绘画等,从有限的空间到无限的情感
	- 不规则的珍珠
- 特征:
	- 从中世纪教会音乐调式,逐渐确立调式体系(巴赫)
	- 和声功能通过通奏**低音**呈现  有了根(低音)->界定和声的功能
	- **器乐兴起**,有了最早期的曲式结构(比较复杂),旋律的发展开始遵循主题与变奏
		- 合唱一般就4个声部,交响乐就多得多
		- 只有一个主部主题,然后分列,展开->让人们的音乐创作有了规律可循
	- 旋律鲜明,节奏明确
	- 音乐对比更加清晰
	- 声乐体裁,器乐体裁都得到了空前发展
		- 歌剧
		- 协奏曲,独奏曲,奏鸣曲
