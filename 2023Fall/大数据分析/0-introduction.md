# 数据挖掘

- 给定大量数据，发现以下模式
	- 有效性
	- 可用性
	- 出乎意料
	- 可理解性

## 常见任务

- 描述类方法
	- 找出人类可理解的模式来描述数据
	- 例：聚类
- 预测类方法
	- 使用一些变量来预测未知的或者未来的其他 变量的值
	- 例：推荐系统

### 操作

1. 收集
2. 准备：数据不为空
3. 表示：要对数据预处理
4. 模型：参数
	- 模型训练：x = a +by，在数据上拟合出来a、b是多少
5. 原因：验证
6. 可视化


### 挑战

1. 可扩展性
2. 数据流：数据量很大
3. 内容
4. 质量：计算结果的质量要求
5. 使用：可用，使用不能太复杂

### 数据形态

1. Ontologies:源数据
2. Structed:结构化
3. Networks:网络
4. Text:文本
5. Multimedia:流媒体
6. Signals:信号
