# 3. C与C++的关系
1. C++完全包含C 语言成分，支持 C 所支持的全部编程技巧（C的超集），C是建立C++的基础
2. 任何 C 程序都能被 C++ 用基本相同的方法编写，并具备相同的运行效率和空间
3. C++还引入了重载、内联函数、异常处理等功能，对 C 中过程化控制及其功能进行了扩充
4. 此外，C++还添加了 OOP 的完全支持
